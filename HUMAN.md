# About Hilal Browser

Hilal Browser is built by Egehan Kahraman, a self-taught developer from Kahramanmaraş, Türkiye.

The project started as Huma Browser and has been under active development since 2026. It is a personal project, not a startup or an organisation.

## What is AI-generated in this repo

This project uses AI coding assistants, including Claude, for:

- Drafting documentation
- Generating boilerplate shell scripts
- Code review and refactoring suggestions

All patches applied to Firefox and all architectural decisions are written or reviewed and approved by the maintainer. The AI-generated parts are scaffolding; the browser customisations are reviewed before they land.

## What is not AI-generated

- The `.patch` files under `changes/`, which are unified diffs against Firefox source
- The branding assets in `changes/browser/branding/hilal/`
- The patch-plus-overlay model, the gitignored `engine/` checkout, and the upstream-first workflow
- The `FIREFOX_COMMIT` and `upstream.lock` pin for Firefox `153.0a1` at `e28b34ab33dbf49364999070168cbb7e11e8e5bd`
